# G-Rump Marketing Site

Static landing and pricing page for G-Rump. Open `index.html` in a browser or serve from any static host (e.g. Vercel, Netlify).

## Sections

- **Hero**: Value proposition and CTA
- **Features**: Architecture, multi-agent codegen, Claude Code quality
- **Pricing**: Free, Pro, Team tiers
- **Links**: Sign up points at web app (configure URL for production)

## Production

Update the app links in `index.html` to match your real deployment:
- Web app: `https://app.g-rump.com`
- GitHub: `https://github.com/Aphrodine-wq/G-rump.com`
