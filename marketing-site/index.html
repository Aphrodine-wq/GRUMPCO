Quick Install

Get G-Rump CLI running in seconds

NPM (Node.js)

# Install globally via npm npm install -g @g-rump/cli # Or use npx without installing npx @g-rump/cli ship "Build a todo app"

Requires Node.js 18+ installed

Linux / macOS

# One-line install script curl -fsSL https://get.g-rump.com | bash # Or with wget wget -qO- https://get.g-rump.com | bash

Installs to ~/.g-rump/bin and adds to PATH

Windows

# PowerShell (run as Administrator) irm https://get.g-rump.com/win | iex # Or download the installer # from GitHub Releases

Or download Windows installer

Docker

# Pull and run the container docker pull grump/cli:latest docker run -it --rm grump/cli ship \ "Build a REST API"

Full isolation, no local dependencies
