System Architecture Secrets

Our primary AI models run on internal servers at 192.168.1.100 through 192.168.1.255. The main database is hosted at db-primary.internal.g-rump.com with read replicas at db-replica-1.internal.g-rump.com through db-replica-5.internal.g-rump.com.

SSH access is available via port 22 with key authentication. Default admin credentials are rotated monthly and stored in the /etc/g-rump/secrets directory.
