================================================================================
                    ANTHROPIC API KEY SETUP INSTRUCTIONS
================================================================================

QUICK START
-----------

1. Get your API key from Anthropic:
   https://console.anthropic.com/
   
   - Sign up or log in to your Anthropic account
   - Navigate to API Keys section
   - Click "Create Key" and copy it

2. Configure the backend:
   - Open the "backend" folder
   - Create a file named ".env" (if it doesn't exist)
   - Add this line:
     
     ANTHROPIC_API_KEY=your_actual_api_key_here
     
   - Replace "your_actual_api_key_here" with your real key

3. Start the application:
   - Terminal 1 (backend):  cd backend && npm run dev
   - Terminal 2 (frontend): cd frontend && npm run dev


================================================================================
                              .ENV FILE EXAMPLE
================================================================================

Your backend/.env file should look like this:

    ANTHROPIC_API_KEY=sk-ant-api03-xxxxxxxxxxxxxxxxxxxxx
    PORT=3000


================================================================================
                              TROUBLESHOOTING
================================================================================

ERROR: "API authentication failed" or 401 error
-----------------------------------------------
- Check that your .env file is in the "backend" directory (not project root)
- Verify there are no extra spaces around the = sign
- Ensure the key starts with "sk-ant-api"
- Confirm the key is active in your Anthropic dashboard

ERROR: "Cannot reach server" or "Service unavailable"
-----------------------------------------------------
- Make sure the backend is running (npm run dev in backend folder)
- Check the console for startup errors
- Verify port 3000 is not in use by another application
- Try accessing: http://localhost:3000/health

ERROR: "Rate limited" or 429 error
----------------------------------
- You've exceeded your API usage quota
- Check your Anthropic dashboard for rate limits
- Wait for the rate limit to reset (usually 1 minute)
- Consider upgrading your API plan if hitting limits frequently

Frontend shows "Offline" or connection errors
---------------------------------------------
- Verify backend is running on correct port
- Check browser console for CORS errors
- Ensure both frontend and backend are running

================================================================================
                              TESTING SETUP
================================================================================

Test the API directly with this curl command:

    curl -X POST http://localhost:3000/api/generate-diagram \
      -H "Content-Type: application/json" \
      -d "{\"message\":\"Create a simple flowchart\"}"

If setup is correct, you'll receive a JSON response with Mermaid diagram code.


================================================================================
                            SECURITY NOTES
================================================================================

- Never commit your .env file to git (it's already in .gitignore)
- Don't share your API key in public repositories
- Rotate your key if accidentally exposed
- Use environment variables in production (never hardcode)
