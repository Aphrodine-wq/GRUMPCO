## Build from Source

If you prefer to build G-Rump from source:

```bash
# Clone the repository
git clone https://github.com/Aphrodine-wq/G-rump.com.git
cd G-rump.com

# Install dependencies
npm install

# Build all components
npm run build

# Or run in development mode
npm run dev
```

See the [Building Guide](/guide/building) for detailed instructions.
