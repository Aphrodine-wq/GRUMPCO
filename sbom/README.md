# SBOM Directory

This directory contains Software Bill of Materials (SBOM) files generated by the `npm run sbom` command.

## Contents

- `*-sbom-*.json` - CycloneDX format SBOM files for each workspace
- `manifest.json` - Index of all generated SBOMs

## Generation

```bash
# Generate SBOMs for all workspaces
npm run sbom
```

## Format

SBOMs are generated in [CycloneDX](https://cyclonedx.org/) 1.5 format, which is compatible with:

- OWASP Dependency-Track
- Snyk
- GitHub Dependency Graph
- Most vulnerability scanning tools

## CI Integration

SBOMs are automatically generated in CI and attached as build artifacts.
