# G-Rump VS Code Extension

Placeholder for a VS Code extension that connects to the G-Rump web API for chat, codegen, and diagram generation.

## Planned features

- Command palette: "G-Rump: New from description"
- In-editor chat panel
- Generate PRD / architecture from selection
- Push generated code to GitHub via API

## Setup

1. Configure `VITE_API_URL` or extension settings to point at your G-Rump API.
2. Run from `integrations/vscode-extension` when implemented.
